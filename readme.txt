Create amazing presentations with Wordpress and Impress.js

== Description ==

Quicky create beautiful presentations in 3 dimentions with help of impress.js and Wordpress. Allows you to use present your posts, pages, media or the Impress slides with the power of Impress.js written by Polish developer Bartek Szopka.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload 'impress' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. A new menu item called "Impress" is now added to your admin menu. Choose what and where to show your slides from the "Settings" section under the new "Impress" item. Then start adding new "Impress" slides blocks.
